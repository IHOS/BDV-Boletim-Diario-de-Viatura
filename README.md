# BDV Digital - Boletim Diário de Viatura

Aplicação web responsiva para digitalização do **Boletim Diário de Viatura (BDV)**, utilizado pela Seção de Transporte do Batalhão de Polícia de Choque. O sistema permite preencher os dados da viatura, serviço, checklist obrigatório, alterações na carga/descarga e gerar automaticamente um PDF no modelo do documento original.

## Funcionalidades

- Formulário responsivo para preenchimento do BDV
- Avisos obrigatórios no topo do sistema
- Cadastro de dados da viatura e serviço
- Controle de quilometragem e horário
- Checklist obrigatório com marcação automática
- Registro de alterações na carga e descarga
- Campos de responsável, sala de meios e STM
- Assinatura digital opcional via canvas
- Geração automática de PDF
- Layout preparado para impressão em A4

## Tecnologias Utilizadas

- HTML5
- TailwindCSS
- JavaScript puro
- html2pdf.js
- html2canvas
- jsPDF
